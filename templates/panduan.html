Panduan Umum dan Tahapan Penginputan Proposal

Tentang

Website ini merupakan wadah untuk memproses proposal di lingkungan Fakultas Sains dan Teknologi Universitas Sultan Syarif Kasim Riau. Pada website ini proposal anda akan melalui tahapan-tahapan berikut :

  1. Pembuatan akun
    1. Pilihlah Jurusan atau Organisasi Anda
    2. Pilih Buat Akun
    3. Isi form sesuai data diri anda
    4. Klik Submit
    5. Silahkan tunggu persetujuan untuk akun anda dari WD 3. Pemberitahuan akan dikirimkan melalui email
    6. Setelah mendapat pemberitahuan melalui email, akun anda dapat digunakan sebagaimana mestinya
  2. Penginputan proposal
  3. Pemeriksaan oleh Kepala Program Studi anda bila anda berasal dari HMJ, sedangkan jika anda berasal dari SEMA dan DEMA proposal anda akan diperiksa langsung oleh Wakil Dekan 3.
  4. Setelah melalui pemeriksaan proposal oleh Kepala Program Studi akan terdapat 2 hasil yaitu revisi dan komentar positif, jika anda mendapatkan komentar positif maka proposal anda akan dilanjutkan dan diperiksa oleh Wakil Dekan 3.
  5. Hasil dari Wakil Dekan 3 ini merupakan hasil akhir diterima atau tidaknya proposal anda.
  6. Untuk semua proses diatas anda akan mendapatkan pemberitahuan melalui email yang anda gunakan saat pembuatan akun.

Panduan Penginputan Proposal

  1. Sebelumnya sediakan 2 file untuk di upload, file proposal anda dan file yang hanya berisi RAB dari proposal anda
  2. Masuk ke website pengajuan proposal
  3. Pilihlah dari instansi mana anda berasal
  4. Login ke akun anda atau buat akun terlebih dahulu jika anda belum memiliki akun
  5. Masuk ke menu penginputan proposal
  6. Masukkan data anda sesuai kolom yang telah disediakan
  7. Upload file yang telah anda siapkan
  8. Klik submit, saat semua penginputan selesai.
  9. Anda dapat melakukan pengeditan data melalui menu edit pada halaman penginputan proposal.
